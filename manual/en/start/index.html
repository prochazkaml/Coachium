Familiarising yourself with the Coachium environment

If you have a device connected (or have run the program in guest mode), Coachium will open its main interface. It is divided into several sections, which are described below.

Status bar

Used to display the current state of the program - connection and disconnection of devices, whether recording is in progress or finished, whether you are selecting a data area, etc.

Also includes a button to disconnect from the device or to reconnect to the device.

Action panel

Contains all the options available to you - loading and saving files, starting and managing records, data display settings, etc.

If the button is completely black, you can safely click on it. If it's more greyed out, it's inactive and clicking it will not perform anything.

You can learn more about these options in the following help chapters.

Main area

Takes up most of the screen. This is where the recorded data will be displayed if you record any (either as a graph or a table, you can choose).

Side panel

It shows all the available ports that the device has.

If a sensor is connected to a port, that port will "light up", displays basic information about the sensor and periodically updates with the current sensor value.

By clicking on the connected sensor, you can reset the sensor or return it to its original value.

Note: this panel is hidden in guest mode.
