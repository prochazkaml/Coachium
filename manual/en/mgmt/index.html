Records Management

Coachium allows you to edit or reorganize records after they are created. These actions are performed using the buttons on the action bar.

Switching between records

If you create multiple records, you will probably want to go back to previous records. To do this, you can use the "View previous workbook record" or "View next workbook record" ( and ) buttons.

Rename a record

If you have a record currently selected that you want to rename, click the "Rename this record" button () to display a text box pre-filled with the current name of the selected record. You can modify it and confirm your changes by clicking the "Rename" button.

Rearrange records

If you find yourself in a situation where you've accidentally created records in the wrong order, don't worry. Clicking the "Manage Records" button () will bring up a window listing all of your records. From there, you can click on a record and use the arrows ( and ) in the bottom right corner of the window to move it up or down in order.

You can also rename a record in this window. The current name of the selected record will automatically appear in the text box in the lower left corner of the window when you select it in the list. You can edit and sub-edit it (by clicking on ).

Viewing information about the record

If you need to view information about the currently selected record and view its setting parameters or sensors used, you can find it in the "Record Details" window. This window is displayed when you click on the "Show details of this record" button ().

Delete record

If the currently selected record has failed, you can delete it. Click the "Delete this record from the workbook" button (), confirm that you are sure, and the record will be gone forever.

WARNING: This operation is irreversible! Make sure you delete the right record!

Delete all records

If your work really failed and you want to start over, click the "Discard all records and create a new record book" (), button to delete all your work and start with a clean record book.

WARNING: This operation is irreversible! Be sure to save your work, at least just in case! See the following sections for more information on saving and loading your work.
