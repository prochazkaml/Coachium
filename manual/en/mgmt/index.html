Managing captures

Coachium allows you to alter or reorganize captures after they are created. These actions are performed using the buttons on the action bar.

Switching between captures

If you create multiple captures, you will probably want to go back to the previous ones. To do this, you can use the "Show the previous capture" or "Show the next capture" buttons ( and ).

Rename a capture

If you have a capture currently selected that you want to rename, click the "Rename this capture" button () to display a text box pre-filled with the current name of the selected capture. You can modify it and confirm your changes by clicking the "Rename" button.

Rearrange captures

If you find yourself in a situation where you've accidentally created captures in the wrong order, don't worry. Clicking the "Manage captures" button () will bring up a window listing all of your captures. From there, you can click on a capture and use the arrows ( and ) in the bottom right corner of the window to move it up or down in order.

You can also rename captures in this window. The current name of the selected capture will automatically appear in the text box in the lower left corner of the window when you select it in the list. You can edit it and confirm it by clicking on .

Display information about the capture

If you need to view information about the currently selected capture, view its setup parameters or see the sensors used, you can find it in the "Capture properties" window. This window is displayed when you click on the "Display this capture's details" button ().

Delete a capture

If you are not satisfied with the currently selected capture, you can delete it. Click the "Delete this capture" button (), confirm that you are sure and the capture will be gone forever.

WARNING: This operation is irreversible! Make sure you are deleting the correct capture!

Delete all captures

If all your work did not come out the way you wanted to and you want to start over, click the "Remove all captures and create a new workbook" button (), to delete all your work and start with a clean workbook.

WARNING: This operation is irreversible! Be sure to back up your work, just in case! See the following sections for more information on saving and loading your work.
