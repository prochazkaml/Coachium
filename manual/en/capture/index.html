Creating a capture

Coachium is primarily designed to capture data and work with it.

If you want to create a capture, you need to have at least one sensor plugged into your device (you can tell if at least one of the otherwise grey blocks on the side panel lights up with a different color). If you meet this condition, you can click on the "Add a new capture to your workbook" button ().

The "Capture setup" window will appear, where you can select which sensors you want to capture from. You can also set the name of this capture, at what frequency the device will capture and for how long. First, however, you must select a capturing mode - either standard mode or X-Y mode.

Standard mode

In standard mode, any number of sensors can be selected (within the capabilities of the hardware) to be captured simultaneously and displayed along the Y axis as a function of time.

X-Y mode

In X-Y mode you must select exactly 2 sensors, one for each axis (X, Y). Useful for comparing different quantities where there is no need to display time (e.g. current-voltage characteristics of a component).

Triggering

An optional option is the so-called triggering, which will wait for a condition to be met before the capture starts. To define this condition, simply drag one of the connected sensors to the lower right area of the window. If you do that, options will appear below it where you can set the condition to trigger the start of the capture.

Start and stop the capture

If no error message appears at the bottom of the window, it means that your settings are valid and you can click the "Start" button to start the capture.

The capture will run for exactly the specified time. However, if you need to end the capture prematurely, you can click the "Stop the current capture" button ().
