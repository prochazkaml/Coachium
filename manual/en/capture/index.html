Creating a record

Coachium is primarily designed to capture data and work with it.

If you want to create a record, you need to have at least one sensor plugged into your device (you can tell if at least one of the otherwise grey blocks on the side panel lights up with a different color). If you meet this condition, you can click on the "Add a new record to the workbook" button. ().

The "Record Settings" window will appear, where you can select which sensors to use for the record. You can also set the name of this record, at what frequency the device will record and for how long. First, however, you must select a recording mode - either standard mode or X-Y mode.

Standard mode

In normal mode, any number of sensors can be selected (within the capabilities of the hardware) to be recorded simultaneously and displayed on the Y-axis as a function of time.

However, it is also possible to view only some of the sensors later, or to generate an X-Y plot with the two selected sensors that were used during the recording.

X-Y mode

In X-Y mode you can select just 2 sensors, one for each axis (X, Y). Useful for comparing different quantities where there is no need to display time (e.g. volt-ampere characteristic of a component).

Conditional recording

An optional option is the so-called conditional recording, which waits for a condition to be met before starting the recording. To define this condition, simply drag one of the connected sensors to the lower right area of the window. After dragging it, options will appear below it where you can set the condition to trigger the recording.

Start and stop recording

If no error message appears at the bottom of the window, it means that your settings are valid and you can click the "Start" button to start the recording.

The recording will run for exactly the specified time. However, if you need to end the recording prematurely, you can click the "Stop recording" button ().
